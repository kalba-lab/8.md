# 8.md

Simple .md viewer for macOS.

## Features

- Markdown rendering with Mermaid diagrams
- Drag & drop files
- Zoom (⌘+/⌘-/⌘0)

## Requirements

- macOS 13.0+

## Install

Download from [kalba.dev/8md](https://kalba.dev/8md)
