# 8.md

Simple .md viewer for macOS.

## Features

- Markdown rendering with Mermaid diagrams
- Drag & drop files
- Zoom (⌘+/⌘-/⌘0)

## Requirements

- macOS 13.0+

## Install

[Download latest release](https://github.com/kalba-lab/8.md/releases/latest)
